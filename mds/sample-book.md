# ようこそ、Bookshelf4MDへ！

これはMarkdownファイルを「本棚」のように閲覧できるWebアプリケーションです。

## 特徴

このアプリケーションには以下の特徴があります：

- **本棚風表示**: カード形式で美しく表示
- **リスト表示**: シンプルな一覧表示
- **サイドバー目次**: 左側に目次を表示
- **進捗表示**: 読書の進捗をパーセンテージで表示
- **コードコピー**: ワンクリックでコードをコピー

## ページ分割のデモ

このブックは`##`見出しで自動的にページ分割されています。

## コードブロックのデモ

以下はPythonのコード例です。右上の「コピー」ボタンをクリックするとコピーできます。

```python
def greet(name):
    """挨拶を返す関数"""
    return f"Hello, {name}!"

# メイン処理
if __name__ == "__main__":
    message = greet("Bookshelf4MD")
    print(message)
```

JavaScriptの例：

```javascript
// 配列の合計を計算
function sumArray(arr) {
    return arr.reduce((sum, num) => sum + num, 0);
}

const numbers = [1, 2, 3, 4, 5];
const total = sumArray(numbers);
console.log(`合計: ${total}`);
```
